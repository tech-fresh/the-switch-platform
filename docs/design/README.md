## Current MVP direction

- Streamlined Mission Control dashboard.
- Five-item student navigation: Dashboard, Subjects, Exams, Progress, Profile.
- Practice lives inside Subjects.
- AI is future scope, not MVP.
- Daily motivation is included in the MVP.
- Visual style is vibrant, premium, accessible, and uncluttered.
- Colour work must use official Switch brand colours and accessible semantic colours.
