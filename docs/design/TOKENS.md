# Tokens

Use existing Switch brand tokens where available.

## Groups

- Colour
- Typography
- Spacing
- Radius
- Shadow
- Motion

## Colour groups

- Brand
- Information
- Success
- Warning
- Error
- Neutral

## Spacing

Use the 8-point spacing system: 8, 16, 24, 32, 40, 48, 64.

## Rule

Do not hardcode random colours or spacing values in new UI work. Reuse approved tokens.
