# The Switch Platform — User Journeys

## New learner

1. Open landing page.
2. Sign up.
3. Complete onboarding.
4. Land on dashboard.
5. Continue first subject.
6. Complete first practice activity.
7. See progress update.

## Returning learner

1. Open dashboard.
2. Read daily motivation.
3. Continue learning.
4. Complete practice or exam task.
5. Review progress.

## Exam preparation

1. Open Exams.
2. Choose subject.
3. Choose board.
4. Choose tier.
5. Choose paper.
6. Start exam.
7. Submit and review results.

## Progress review

1. Open Progress.
2. See predicted grade and target grade.
3. Review topic mastery.
4. Identify weakest topic.
5. Return to subject practice.

## Design rule

Every journey must have a clear next step and avoid unnecessary decisions.
